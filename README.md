# myVideo

使用react-native开发的一个视频应用，目前只实现了安卓版本，项目还在完善当中。。。

### 目前实现的功能

+ 电影列表展示
+ 视频播放
+ 友盟统计
+ 热更新
+ 注册登录
+ 收藏评论

### api说明

视频数据是通过爬虫获取的，由另一个后台项目提供
